Что это такое?

molva.spb.ru — это агрегатор общественного мнения, высказанного через Twitter.

For english version see this post.

Слышать всё.

Агрегатор получает сообщения из Twitter через Twitter API — в реальном времени.

Узнавать каждое слово.

С помощью технологии Яндекса — Томита-парсера — агрегатор узнает каждое слово в русском языке, сохраняя только самые важные.

Понимать каждое слово.

Агрегатор ведет учет контекста для каждого встреченного слова — с какими словами оно встречается? Какие слова используют люди в ответных сообщениях?

Контекст слов сравнивается, чтобы найти наиболее близкие сейчас друг другу слова.

Следить за словом времени.

Агрегатор следит за набирающими популярность явлениями — и показывает их на своей главной странице.

Учиться говорить по-русски.

Разрозненные слова агрегатор превращает в сочетания, подбирает им форму, пытается составить из них предложения.

Задать направление.

Каждое предложение можно уточнить поиском по интернету и новостям — через любой поисковик.
